# Sorting Algorithm Visualizer

A Python application that visualizes how sorting algorithms work step by step using animated bar charts.

## Description

This project demonstrates how sorting algorithms operate by visualizing the sorting process in real time. Random numbers are displayed as vertical bars, and the program updates the bars as the algorithm compares and swaps elements.

The goal of this project is to help users understand how sorting algorithms work internally through visual animation.

## Features

- Visualizes the Bubble Sort algorithm
- Displays numbers as animated bar charts
- Shows step-by-step sorting process
- Generates random data automatically

## Technologies Used

- Python
- matplotlib
- random

## How to Run

1. Install the required library

```bash
pip install matplotlib
```

2. Run the program

```bash
python sorting_visualizer.py
```

## Example

The program generates random numbers and displays them as bars.  
As the sorting algorithm runs, the bars move until the list becomes sorted.

## Purpose

This project was created to practice algorithmic thinking, Python programming, and data visualization while gaining a better understanding of how sorting algorithms work.
